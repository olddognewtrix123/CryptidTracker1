## Installation & Setup

### 1. Install Java 11

**Windows/Mac:**
- Download from [Oracle](https://www.oracle.com/java/technologies/javase/jdk11-archive-downloads.html) or [OpenJDK](https://adoptium.net/temurin/releases/?version=11)

**Linux:**
```bash
sudo apt install openjdk-11-jdk  # Ubuntu/Debian
```

### 2. Verify Java Version
```bash
java -version
# Should show version 11.x.x
```

### 3. Clone the Repository
```bash
git clone https://github.com/yourusername/yourproject.git
cd yourproject
```

### 4. Open in NetBeans
1. Open NetBeans IDE
2. File → Open Project
3. Navigate to the cloned directory
4. Select the project and click "Open"

### 5. Configure Java 11 in NetBeans
1. Right-click the project → Properties
2. Go to "Libraries" category
3. Ensure Java Platform is set to JDK 11
